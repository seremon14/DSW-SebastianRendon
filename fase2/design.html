Etiquetas semánticas, Listas y enlaces

Este sitio tiene como propósito el aprendizaje y utilización de manera adecuada de las etiquetas semánticas, Listas y enlaces


Etiquetas semánticas

Las etiquetas semánticas estructurales nos sirven para que los motores de búsqueda, o cualquier otro mecanismo automático que lea un sitio web, sepa con exactitud qué partes de su contenido corresponden a cada una de las partes típicas de un sitio.
Generalmente, en cualquier documento tenemos una cabecera, un cuerpo y un pie de página, elementos que definen la estructura representados por diversas etiquetas.
De este modo, HTML5 tiene una etiqueta HEADER, que debe englobar aquellos elementos correspondientes a la cabecera de la página. Tiene un FOOTER, que debe englobar todas las etiquetas que componen el pie. Luego, para el cuerpo realmente no existe
una etiqueta en concreto, sino que se representará por diversas otras como SECTION, ARTICLE o ASIDE. Existirán además otras etiquetas que representan elementos típicos como la barra de navegación, etiqueta NAV.

Miguel Angel Alvarez. (2012). Etiquetas semánticas del HTML5. https://desarrolloweb.com/articulos/etiquetas-semanticas-html5.html

Listas

Las listas se utilizan para agrupar información parecida o relacionada a una web. Permiten a Los visitantes tener acceso a grupos de información de manera rápida y sencilla. Puedes incluir cualquier cosa en las listas, desde enlaces hasta en operaciones de cosas.

Listas sin enumerar

Contiene los elementos y listados, precedidos de la viñeta. Para crearla se necesitan 3 elementos en el siguiente orden:

  • <ul> indica el comienzo de una lista sin enumerar
  • <li></li>encerrando cada elemento o información listada
  • </ul> indica el final de la lista sin enumerar

Listas numeradas

se trata de listas en las que aparece el número de orden delante de cada elemento. Se componen de tres elementos:

  • <ol> indica el comienzo de una lista numerada
  • <li></li>encerrando cada elemento o información listada
  • </ol> indica el final de la lista numerada

Prescott, P. (2015). HTML 5. Babelcube Inc. https://books.google.es/books?hl=es&lr=&id=BuyNCgAAQBAJ&oi

Enlaces

Los enlaces constituyen la esencia del lenguaje Html y de las páginas web. La riqueza del hipertexto es la que permite tejer esta red gigantesca que compone la web.
La etiqueta <a> .... </a> inserta un enlace. Su sintaxis básica es:
<a href="destino_del_enlace"> Texto del enlace </a>

El destino del enlace puede ser:

  • Un lugar de la página en curso
  • Otra página del sitio web
  • Algún lugar de otra página del sitio web
  • Una página de otro sitio existente en la web
  • Una dirección de correo electrónico
  • Un archivo para descargar

Van Lancker, L. (2012). HTML5: Los fundamentos del lenguaje. Ediciones ENI. https://books.google.es/books?hl=es&lr=&id=9ZeN3sjDuNkC&oi
